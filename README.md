# Waste Classification using YOLOv5

## Overview
This project detects and classifies waste items such as plastic, metal, paper, glass, and organic waste using the YOLOv5 deep learning model.

## Technologies Used
- Python
- YOLOv5
- OpenCV
- PyTorch
- Google Colab

## Dataset
- Custom waste dataset
- Labeled using Roboflow / LabelImg

## How It Works
1. Input image/video
2. YOLOv5 detects waste objects
3. Outputs class labels with bounding boxes

## Applications
- Smart waste management
- Recycling automation
- Environmental monitoring
